* Overview

RubyGems Integration for Concourse Pipelines.

* Usage

Add the resource to your pipeline's ~resource_types~ (requires Concourse 0.74.0+)

#+BEGIN_SRC yaml
  resource_types:
    - name: rubygems
      type: docker-image
      source:
        repository: resource/rubygems
#+END_SRC
